# universe-app

Scaffold mínimo de um app React + Vite que documenta o Universo Zero-Entropy.

Instalação e execução:

```bash
cd "D:/IA/UBUNTU IA/universe-app"
npm install
npm run dev
```

Abra http://localhost:5173

Arquivos importantes:
- `public/manifest.json` — manifesto do universo usado pela aplicação
- `src/pages` — páginas por cluster e front-end
